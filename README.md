# Clone do "Tab News"

Aplicação web desenvolvida com **Next.js** para estudos no curso **CURSO.DEV**.

O projeto tem como objetivo praticar **desenvolvimento frontend moderno** e **deploy em ambiente real** utilizando a Vercel.

---

- Trabalhar com **Next.js**
- Estruturar um projeto frontend moderno
- Realizar **deploy contínuo** com a Vercel
- Praticar versionamento com Git e GitHub

## Como Executar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/RichardVes/tab-news-frase.git
   ```
2. Acesse a pasta do projeto:
   cd tab-news-frase
3. Instale as dependências:
   npm install
4. Inicie o servidor de desenvolvimento:
   npm run dev
5. Acesse no navegador:
   http://localhost:3000
