#PROTOCOLOS
São padroes de regras que definem a comunicação entre duas partes, que define como se inicia, como se preenche as informações e como a informação vai voltar

- HTTP
  Hypertext Transfer Protocol
  É um protocolo que tem referencias para outros documentos, define regras para essas transferencias
- FTP
  File Transfer Protocol
  É um protocolo dedicado para transferencias de arquivos
- SMTP
  Simple Mail Transfer Protocol
  É um dos padrões de transferencia de mensagens de emails
- TCP
  Error Recover
  Recuperação de erros
- IP
  Internet Protocol
- UDP
  User Datagram Protocol
  Datagram é um pacote alto contido, não precisa ser validado
